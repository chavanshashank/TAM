Technoarena 2015

Code Forces


If you think you are really passionate for C programming then this is exactly the event that you are looking for. This event gives all the C/C++ programmers a platform to showcase their coding skills. Code Forces will give participants an excellent opportunity to evaluate their C/C++ coding skills and competing with others will be a pleasant experience to test yourself. Participants would be tested on grounds of aptitude test base on C/C++ programming, debugging & efficiency. Participate and test your passion towards C/C++ programming.

Rules

  1. This event consists of two rounds.
  2. Use of external devices such as pen drive, DVD, Mobile is strictly prohibited.
  3. Additional rules will be announced on event.
  4. Participation certificate will be given to all the participants.

Structure

  1. There will be 2 rounds
  2. Round 1: Aptitude Test
    a) This round consist of 50 multiple choice questions.
    b) These questions are totally based on knowledge of C Programming Language only.
    c) There is negative marking.
        i) For each correct answer there is +2 marks.
        ii) For each wrong attempt there is -1 mark.
        iii) No mark for no attempt.
    d) Minimum top 15 participants will be selected for next round.
  3. Round 2: Programming
    a) In this level all participants have to implement a programs in c or c++ language according to provided questions.
